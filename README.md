# REFcompliancereport
A shiny app that checks a Figshare batch download for REF compliance

Build file
1.	Batch download all items from repo, both public and private
2.	Include only items that have an embargo date set in the future / permanent embargo
3.	Include only journal articles, chapters, and conference papers
4.	Display 
  •	Item_id
  •	Author
  •	Title
  •	DOI
  •	Version
  •	Handle  
5.	Sort by embargo date
